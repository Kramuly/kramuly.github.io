# kramuly.com

Single-page holding/landing site for Kramuly — agentic commitment intelligence.

- One self-contained `index.html` (inline CSS, minimal vanilla JS, Google Fonts only — no build step).
- Served via GitHub Pages at **https://kramuly.com** (custom domain set in `CNAME`).

## Deploy
Any push to `main` republishes automatically. To preview locally just open `index.html` in a browser.
